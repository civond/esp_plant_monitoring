
<h1>LeafLogic (ESP32-based Plant Monitoring)</h1>
   
<div>

  <ul>
    <li>ESP-WROOM-32 Dev kit</li>

    <li>Adafruit MicroSD SPI Breakout Board</li>

  </ul>

  <h3>Libraries:</h3>
